NAME

copy_shash - copy a shash

SYNOPSIS

#include <hash.h>

shash copy_shash (pool, shash);

DESCRIPTION

Copy a shash into a new pool. This function copies the keys and values, but if keys and values are pointers, then it does not perform a 'deep' copy.
